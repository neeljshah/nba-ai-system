  <name>Task 1: DeepSORT tracker and coordinate computation</name>
  <files>tracking/tracker.py</files>
  <behavior>
    - TrackedObject is a dataclass with: track_id (int), object_type (str 'player'|'ball'), cx (float), cy (float), bbox (tuple), confidence (float), frame_number (int), timestamp_ms (float), velocity_x (float), velocity_y (float), speed (float), direction_degrees (float)
    - ObjectTracker.__init__(max_age: int = 30, n_init: int = 3) initializes DeepSort with these params
    - ObjectTracker.update(detections: List[Detection], frame: np.ndarray, frame_number: int, timestamp_ms: float) returns List[TrackedObject]
    - Velocity is computed as (current_cx - previous_cx) * fps per second — requires storing previous positions by track_id
    - ObjectTracker stores previous_positions: dict[int, tuple[float, float]] between calls
    - speed = sqrt(velocity_x^2 + velocity_y^2)
    - direction_degrees = degrees(atan2(velocity_y, velocity_x)), normalized to [0, 360)
    - On first frame for a new track_id, velocity_x = velocity_y = speed = 0.0, direction_degrees = 0.0
    - DeepSort track.get_det_class() is not reliable — use object_type from the original detection matched by bbox overlap (nearest centroid to track ltwh center)
  </behavior>
  <action>
    Create tracking/tracker.py:
      - Import deep_sort_realtime.DeepSort, numpy, math, dataclasses
      - Define TrackedObject dataclass with all fields listed above
      - ObjectTracker class:
        - __init__: self.deepsort = DeepSort(max_age=max_age, n_init=n_init), self.previous_positions = {}, self.fps = 30.0 (default; updated by set_fps())
        - set_fps(fps: float): sets self.fps
        - update(detections, frame, frame_number, timestamp_ms):
          1. Convert Detection list to DeepSORT input format: list of ([x1,y1,w,h], confidence, class_label) tuples. Width = x2-x1, height = y2-y1.
          2. Call self.deepsort.update_tracks(raw_detections, frame=frame)
          3. For each confirmed track (track.is_confirmed()):
             - Get track_id = track.track_id
             - Get ltwh = track.to_ltwh(); cx = ltwh[0]+ltwh[2]/2, cy = ltwh[1]+ltwh[3]/2
             - Match to original detection by nearest cx/cy to get object_type and confidence (use min distance; default to 'player' if no detection matched)
             - Compute velocity from previous_positions.get(track_id)
             - Build TrackedObject
          4. Update self.previous_positions[track_id] = (cx, cy)
          5. Return list of TrackedObject

    Note: deep-sort-realtime API: DeepSort.update_tracks(detections, frame=frame) where detections is list of ([left,top,w,h], confidence, det_class). Returns list of Track objects.
  </action>
