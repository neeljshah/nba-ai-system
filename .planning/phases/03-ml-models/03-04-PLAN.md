  <action>
    Create models/lineup_optimizer.py implementing ML-05.

    Architecture: LineupOptimizer is a lookup + scoring model, not a traditional
    classifier. It stores a dict of per-player stats (EPA + defensive metrics)
    and computes lineup scores analytically. joblib serializes the whole object
    (including the stats dict) — no separate sklearn estimator needed, but
    BaseModel contract is still followed.

    fit(df: pd.DataFrame) -> LineupOptimizer:
    1. Load per-player EPA by calling PlayerImpactModel.load('player_impact').rank_players()
       If artifact missing (first run), build an empty EPA dict and log warning.
    2. Load defensive pressure stats from DB:
       SELECT track_id, AVG(nearest_defender_dist) as avg_def_dist,
              AVG(ABS(closing_speed)) as avg_closing_speed
       FROM feature_vectors GROUP BY track_id
       Using get_connection() from tracking.database.
    3. Store self._player_stats: dict[int, dict] mapping track_id ->
       {epa_per_100, avg_def_dist, avg_closing_speed}
    4. If no stats from DB: generate synthetic stats for 20 synthetic player IDs
       (track_ids 1-20) with:
         epa_per_100 ~ U(-5, 10)
         avg_def_dist ~ U(50, 250)
         avg_closing_speed ~ U(0, 15)
    5. Return self.

    predict(features: dict) -> dict:
    - features key: 'lineup' = list of 5 track_ids (ints)
    - For each track_id, look up self._player_stats[track_id] or use fallback:
        {'epa_per_100': 0.0, 'avg_def_dist': 150.0, 'avg_closing_speed': 5.0}
    - offensive_gravity = mean(epa_per_100 across lineup) normalized to [0,1]
        using: (mean_epa + 10) / 20  clipped to [0,1]
    - defensive_disruption = mean(avg_closing_speed) / 15.0 clipped to [0,1]
        (avg_closing_speed max ~15 px/s)
    - lineup_score = 0.5 * offensive_gravity + 0.5 * defensive_disruption
    - Return {'offensive_gravity': float, 'defensive_disruption': float, 'lineup_score': float}

    score_lineup(lineup: list[int]) -> dict:
    - Convenience wrapper around predict({'lineup': lineup})

    compare_lineups(lineups: list[list[int]]) -> list[dict]:
    - Score each lineup, return sorted list of {'lineup': list, **scores} by lineup_score desc

    IMPORTANT: self.fit() MUST be called before predict(). Raise RuntimeError
    if predict() called before fit() (check if self._player_stats is None).

    Add __main__ block with --train flag.
